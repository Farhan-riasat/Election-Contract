# install dependencies
Node >8.2 <br>
Truffle Framework (npm install -g truffle) <br>
Ganache <br>
MetaMask  <br>

# installation
git clone https://github.com/farhan-riasat/Election-Contract.git    <br>
cd Election-Contract  <br>
npm install <br>
truffle migrate --reset <br>
npm run dev <br>

# test
truffle test
